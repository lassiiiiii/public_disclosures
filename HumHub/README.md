# HumHub

> HumHub is a free and open-source social network software written on top of the Yii PHP framework that provides an easy-to-use toolkit for creating and launching your own social network.
> -- [Wikipedia](https://en.wikipedia.org/wiki/HumHub)

Homepage: https://www.humhub.com/

Security policy: https://github.com/humhub/humhub/security

## Personal experience
Quick to respond and fix the reported vulnerabilities. Overall a pleasant experience, their contact person was lovely. Minor disagreements about severity score and some CVE/advisory filings, but nothing big.
