# CoolerControl

> CoolerControl is an open-source application for monitoring and controlling supported cooling devices on Linux. It features a system daemon with a built-in Web UI, an optional desktop app, and a comprehensive REST API.
> -- [CoolerControl README](https://gitlab.com/coolercontrol/coolercontrol)

Repo: https://gitlab.com/coolercontrol/coolercontrol

Docs: https://docs.coolercontrol.org/

Security policy: https://gitlab.com/coolercontrol/coolercontrol/-/blob/main/SECURITY.md

## Personal experience

CoolerControl didn't have a security policy when I found vulnerabilities in 2026 January, so I had some issues reporting at first (partly my fault as well). Once I got in touch with the maintainer (Guy Boldon, https://gitlab.com/codifryed), everything went great and he's wonderful to work with. 5/5 would recommend
